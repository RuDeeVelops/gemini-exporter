# Installation Guide

## Quick Start

Follow these simple steps to install the Gemini Chat Exporter extension:

### For Chrome, Edge, and Brave

1. **Download the Extension**
   - Clone this repository or download it as a ZIP file
   - If downloaded as ZIP, extract it to a folder

2. **Open Extension Settings**
   - **Chrome**: Navigate to `chrome://extensions/`
   - **Edge**: Navigate to `edge://extensions/`
   - **Brave**: Navigate to `brave://extensions/`

3. **Enable Developer Mode**
   - Look for a toggle switch labeled "Developer mode" in the top-right corner
   - Click it to enable developer mode

4. **Load the Extension**
   - Click the "Load unpacked" button
   - Navigate to and select the `gemini-exporter` folder
   - Click "Select Folder"

5. **Verify Installation**
   - You should see the "Gemini Chat Exporter" extension appear in your list
   - The extension icon should appear in your browser toolbar

6. **Start Using**
   - Navigate to [Gemini](https://gemini.google.com/)
   - Open any chat conversation
   - Click the extension icon to export!

### For Firefox

1. **Download the Extension**
   - Clone this repository or download it as a ZIP file
   - If downloaded as ZIP, extract it to a folder

2. **Open Debugging Page**
   - Navigate to `about:debugging#/runtime/this-firefox`

3. **Load Temporary Add-on**
   - Click "Load Temporary Add-on..."
   - Navigate to the `gemini-exporter` folder
   - Select the `manifest.json` file
   - Click "Open"

4. **Note for Firefox Users**
   - Temporary add-ons are removed when Firefox is restarted
   - You'll need to reload it each time you restart Firefox
   - For a permanent installation, the extension would need to be signed by Mozilla

5. **Start Using**
   - Navigate to [Gemini](https://gemini.google.com/)
   - Open any chat conversation
   - Click the extension icon to export!

## Troubleshooting Installation

### "Load unpacked" button is grayed out
- Make sure Developer Mode is enabled

### Extension doesn't appear after loading
- Try refreshing the extensions page
- Check the browser console for error messages
- Verify all files are present in the folder

### Extension icon doesn't show in toolbar
- Right-click the toolbar and select "Extensions" or "Manage Extensions"
- Find "Gemini Chat Exporter" and click the pin icon

### Permission errors
- Make sure you're on the correct page (`gemini.google.com`)
- Try reloading the extension
- Refresh the Gemini page

## Updating the Extension

When a new version is released:

1. Download the updated version
2. Go to your browser's extension page
3. Click "Reload" or "Update" next to the Gemini Chat Exporter extension
4. Or remove the old version and load the new one

## Uninstalling

To remove the extension:

1. Go to your browser's extension page
2. Find "Gemini Chat Exporter"
3. Click "Remove" or the trash icon
4. Confirm the removal
